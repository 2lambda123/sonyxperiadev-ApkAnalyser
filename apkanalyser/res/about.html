Changes since 5.0

  • Support UML package/class diagrams
  • Highlight dependecies of packages/classes
  • Explore fields/resources/xml in the tree
  • Explore all references as sub nodes
  • Add Preview panel to display contents and diagrams
  • Find labels in resource and const string in code
  • Support Android SDK as class path
  • Support Ice Cream Sandwich

Changes since 4.0

  • List resource IDs and its contents
  • Look up resource references in xml, code or other resource IDs
  • View resource detail, like name, type, languages, configs, references...
  • Find unused resources by resource ID or files
  • Find system resource (@android) references
  • Detect resource potential issues
  • Support Gingerbread

Changes since 3.0

  • Look up field read/write accesses
  • Modify APK file with Davik bytecode injections
  • Uninstall/resign/install/launch APK file
  • Logcat viewer with debug level filters
  • View optimized dex file(odex) dependencies
  • Disassemble, modify and update system odex applications
  • Support odex libraries as classpath
  • Support Froyo

Changes since 2.0

  • Support Android and rename to ApkAnalyser
  • Explore package/classes/methods/fields
  • Look up package/classes/methods references
  • Paint method's call graph
  • Disassemble dalvik method's bytecode with highlighting syntax
  • Loop up dalvik monitor calls (synchronization statements)
  • Decode Android XML files with highlighting syntax
